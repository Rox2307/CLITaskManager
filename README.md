# CLI Command Prompt Task Manager

A command-line tool to manage tasks, with saving, loading, searching, and updating features.

## Features
- Add, update, delete tasks
- Search and filter
- Save/load tasks
- Command-prompt style CLI
- Enable autoupdate

## Installation
1. Clone the repository

## Run program
2. Run "main.py" file
